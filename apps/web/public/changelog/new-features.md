# New features

## Meeting assignment tools
Meetings can now be assigned to the right CRM deal, then resolved into that deal's notes and activity trail.

## Meeting attendee details
Meeting records now show richer attendee context so sales teams can quickly see who joined each call.

## Proposal version history
Proposal detail pages now surface version history and creator context so teams can track proposal updates more clearly.

## Safer deal deletion
Deals now move through a trash flow before permanent deletion, reducing the risk of accidental data loss.

## Revenue reseller view inside main revenue
The main Revenue page now includes the reseller summary cards, product breakdown, deal pricing table, and MRR table under the Reseller tab.

## Billing modal from revenue cells
Missing billing values in the revenue table can now be added from an inline billing button without leaving the Revenue page.

## Catalog tabs across pipeline and revenue
Pipeline and Revenue now share catalog parent tabs and sub-tabs for All, Products, Services, Reseller, and Partnership views.
